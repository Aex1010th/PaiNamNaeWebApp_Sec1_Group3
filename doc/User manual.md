การเข้าสู่ระบบ  
1.การสมัครสมาชิก  
&nbsp;1.คลิ้กที่สมัครสมาชิก บนแถบเมนู  
&nbsp;2.กรอกชื่อผู้ใช้,อีเมล,รหัสผ่านและยืนยันรหัสผ่าน  
&nbsp;3.กดถัดไป  
&nbsp;4.กรอกชื่อ,นามสกุล,เบอร์โทรศัพท์,เลือกเพศ(ชายหรือหญิง)  
&nbsp;5.กดถัดไป  
&nbsp;6.เลือกอัพโหลดรูปภาพบัตรประชาชน กรอกหมายเลขบัตรประชาชนและวันหมดอายุบัตร เลือกอัพโหลดรูปถ่ายใบหน้าตนเอง  
&nbsp;7.กดยอมรับเงื่อนไข  
&nbsp;8.กดสมัครสมาชิก  
  
2.การล็อกอิน  
&nbsp;1.คลิ้กเข้าสู่ระบบ บนแถบเมนู  
&nbsp;2.กรอก ชื่อผู้ใช้ หรือ อีเมล และกรอกรหัสผ่าน  
&nbsp;3.หากยังไม่มีบัญชี กดสมัครสมาชิก  
  
ระบบรายงานปัญหา  
1.การรายงานปัญหาของระบบเว็ปไซต์  
&nbsp;1.ล็อกอินผู้ใช้  
&nbsp;2.เข้าสู่เมนูผู้ใช้งาน:คลิ้กที่แถบชื่อผู้ใช้งาน  
&nbsp;3.เลือกเมนูรายงานปัญหา  
&nbsp;4.กรอกหัวข้อปัญหา,รายละเอียด  
&nbsp;5.เลือกประเภทปัญหา เช่น แอปพลิเคชั่นขัดข้อง แผนที่ขัดข้อง ระบบล่าช้า อื่นๆ  
&nbsp;6.แนบหลักฐาน เช่น ภาพ เสียง วิดิโอ  
&nbsp;7.ส่งรายงาน  
  
2.ระบบคนขับรายงานผู้โดยสาร  
&nbsp;1.ล็อกอินผู้ใช้คนขับ  
&nbsp;2.คลิกเมนูการเดินทางทั้งหมด:เลือกคำขอจองของฉัน  
&nbsp;3.กดดูรายการทั้งหมด  
&nbsp;4.เลือกผู้โดยสาร  
&nbsp;5.กดไอคอนธงสีแดงที่อยู่ข้างชื่อผู้โดยสาร  
&nbsp;6.กรอกหัวข้อปัญหา,รายละเอียด  
&nbsp;7.เลือกประเภทปัญหา เช่น อุบัติเหตุ พฤติกรรมผู้โดยสาร อื่นๆ  
&nbsp;8.แนบหลักฐาน เช่น ภาพ วิดิโอ เสียง  
&nbsp;9.ส่งรายงาน  
  
3.ระบบแอดมินอัพเดตสถานะรายงานปัญหา  
&nbsp;1.ล็อกอินแอดมิน  
&nbsp;2.เข้าสู่เมนูผู้ใช้งาน:คลิ้กที่แถบชื่อผู้ใช้งาน  
&nbsp;3.เลือกเมนู Dashboard  
&nbsp;4.เข้าสู้เมนูแอดมิน:คลิ้กIncident Management  
&nbsp;5.เลือกรายงานที่ต้องการอัพเดท  
&nbsp;6.คลิ้กที่รายงานนั้น  
&nbsp;7.กดเลือกUPDATE STATUS ที่ Admin Response ซึ่งจะมีสถานะคือ Received,In Progress,Resolved,Rejected  
&nbsp;8.กรอกรายละเอียดการตอบกลับ (ไม่บังคับ)  
&nbsp;9.กดSave การตอบกลับ
